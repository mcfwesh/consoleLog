## Installation

### Clone

- Clone this repo to your local machine using `https://github.com/fvcproductions/SOMEREPO`

### Setup

> Run the server

- Run this on the main directory

```shell
$ npm run dev
```

> Run the client

```shell
$ cd client
$ npm start
```
